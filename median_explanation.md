# 中位数（Median）详解

## 什么是中位数？

中位数是统计学中用于描述数据集中趋势的一个重要指标。它表示一组数据按大小顺序排列后，位于中间位置的数值。

## 中位数的特点

1. **不受极端值影响**：中位数对极端值不敏感，当数据中存在异常值时，中位数比平均数更能代表数据的典型水平
2. **稳健性**：中位数是稳健的统计量，对数据分布的偏斜不敏感
3. **直观性**：中位数将数据集分为相等的两部分

## 中位数的计算方法

### 基本步骤：
1. 将数据从小到大排序
2. 确定数据个数n
3. 根据n的奇偶性计算中位数

### 具体公式：
- **当n为奇数时**：中位数 = 第(n+1)/2个数据
- **当n为偶数时**：中位数 = (第n/2个数据 + 第n/2+1个数据) / 2

## 中位数 vs 平均数

| 特征 | 平均数 | 中位数 |
|------|--------|--------|
| 计算方式 | 所有数据之和除以数据个数 | 排序后的中间值 |
| 对极端值敏感度 | 敏感 | 不敏感 |
| 适用场景 | 数据分布均匀时 | 数据有极端值或偏斜时 |
| 稳健性 | 较低 | 较高 |

## 实际应用示例

### 示例1：工资数据
假设一个公司的工资数据为：[3000, 3500, 4000, 4500, 5000, 5500, **100000**]
- 平均数 ≈ 17,000元（受100000影响）
- 中位数 = 4500元（更能反映典型工资水平）

### 示例2：房价数据
房价数据通常右偏（少数高价房拉高平均数），中位数更能反映典型房价

## Python实现详解

我创建的`median_calculator.py`包含以下功能：

### 1. 基础中位数计算
```python
def calculate_median(data):
    # 对数据排序
    sorted_data = sorted(data)
    n = len(sorted_data)
    
    if n % 2 == 1:  # 奇数
        return sorted_data[n // 2]
    else:  # 偶数
        return (sorted_data[n // 2 - 1] + sorted_data[n // 2]) / 2
```

## 使用建议

1. **数据清洗**：计算前确保数据有效，处理缺失值
2. **数据类型**：支持整数和浮点数
3. **异常处理**：程序包含空数据检查
4. **性能考虑**：时间复杂度为O(n log n)，主要来自排序操作
